# 賈維斯超能力入口網站

全新賈維斯 client-facing portal skeleton.

## 本機預覽

```bash
python3 -m http.server 8765 --directory jarvis-superpower-portal
open http://127.0.0.1:8765/
```

## 預計 GitHub Pages 結構

- `/`：主入口
- `/websites.html`：網址列表
- `/file-manager.html`：Google Drive 檔案總管
- `/ppt-catalog/`：HTML PPT 目錄
- `/jarvis-skills.html`：賈維斯技能列表
